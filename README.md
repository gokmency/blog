Minimal blog/portfolio (nav.al aesthetic) powered by **Next.js App Router + Tailwind** and **Hashnode Headless GraphQL**.

## Getting Started

Install deps & run dev:

```bash
npm i
npm run dev
```

Open `http://localhost:3000`.
