Minimal blog/portfolio (nav.al aesthetic) powered by **Next.js App Router + Tailwind** and **Hashnode Headless GraphQL**.

## Getting Started

Install deps & run dev:

```bash
npm i
npm run dev
```

Open `http://localhost:3000`.

## Ask (Gemini)

This project includes a minimal `/ask` page backed by Gemini (server-side).

Set:

```bash
GOOGLE_API_KEY="your_google_ai_studio_api_key"
```

Security note: never hardcode API keys in the repo. If a key was ever pasted into chat or code, rotate it.
